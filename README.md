# Voici comment on utilise le programme:
- Vous ouvrez le code fichier_final.asm sur [RARS](https://github.com/TheThirdOne/rars/releases/tag/continuous).
- Vous modifiez la variable `path` avec le chemin du fichier que vous voulez transformer.
- Vous modifier la variable `path_output` par le nom du fichier que vous voulez avoir en sortie.
- Vous lancer l'execution, vous aurez un menu pour choisir le type de contour.
- Puis vous choisissez la taille maximale d'une ligne.
- Si y a une erreur, vous aurez un message d'erreur.
- Sinon, regardez dans votre dossier vous trouverez le fichier `path_output`.

### A vous de jouer :)
